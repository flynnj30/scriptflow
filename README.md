# ScriptFlow Pro - Smart CRM Workspace

A comprehensive CRM workspace with productivity tools for sales professionals.

## 🚀 Features

- **Smart Workspace**: Tabbed browser with CRM homepage
- **Floating Panels**: Notepad, Call Script, Calculator, Timer
- **Quick Copy Library**: Reusable templates for emails, SMS, scripts
- **Split Screen**: View multiple pages side by side
- **Command Palette**: Keyboard-first navigation (⌘K)
- **Session Recovery**: Restore tabs after browser restart
- **Dark/Light Mode**: Theme support
- **Keyboard Shortcuts**: Power user shortcuts

## 📦 Installation

```bash
# Clone repository
git clone https://github.com/yourusername/scriptflow-pro.git
cd scriptflow-pro

# Install dependencies
npm install

# Start server
npm start
